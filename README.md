# tesla-sales-ml-pipeline
# Tesla Sales & Price Forecasting using Machine Learning

## Overview

This project implements an end-to-end Machine Learning pipeline using Tesla EV deliveries and production data (2015–2025). The workflow covers data preprocessing, exploratory data analysis (EDA), feature engineering, regression modeling, hyperparameter tuning, and time-series forecasting.

## Dataset

**Tesla EV Deliveries & Production Data (2015–2025)**

Source: https://www.kaggle.com/datasets/nalisha/tesla-ea-deliveries-and-production-data20152025

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Linear Regression
* Random Forest Regression
* Hyperparameter Tuning using GridSearchCV
* Feature Importance Analysis
* Time Series Forecasting using ARIMA
* Model Evaluation (MAE, RMSE, R² Score)

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Statsmodels

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Feature Engineering
5. Regression Modeling
6. Hyperparameter Tuning
7. Time Series Forecasting
8. Performance Evaluation

## Results

* Linear Regression achieved an R² score of approximately **0.99**.
* Random Forest Regression achieved an R² score of approximately **0.99**.
* ARIMA was used to forecast future delivery trends.
